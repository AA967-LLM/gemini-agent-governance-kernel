The Authority

Gemini Pro (Weight 3.0) leads synthesis and architecture. Groq (Weight 1.0) provides adversarial validation and security checks. Veto power ensures safety without deadlock.

The Observer

A real-time Flight Recorder (TUI) visualizes agent reasoning. Includes Loop Detection to halt stuck agents and Fuel Gauges for API quotas.

The Civilization

Self-healing capability. If the Council deadlocks, a Mediator Agent spawns autonomously to resolve the conflict using Chain-of-Verification.
